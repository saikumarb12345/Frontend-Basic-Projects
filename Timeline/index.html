Timeline of Linux History

  • 1991 - Linus Torvalds creates the first version of the Linux kernel
  • 1992 - The first official version of Linux, version 0.12, is released
  • 1995 - The first version of Red Hat Linux, a popular Linux distribution, is released
  • 1998 - The Linux Standard Base (LSB) is created to standardize Linux distributions
  • 2007 - Google announces its Android mobile operating system, which is based on the Linux kernel
  • 2015 - The Linux kernel reaches version 4.0
  • 2021 - The Linux kernel reaches version 5.11
